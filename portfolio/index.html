Milford Cinema

This is a cinema website where visitors can browse movies. If they sign up and log in, they can also leave comments and select seats. The technologies used in this web app are React.js, React-Router, React-Redux, Redux-Thunk, REST API, Firebase Realtime Database, Firebase Storage, Google Maps, etc.

Click here to check it out!

Finding Neo

A near-Earth object (NEO) is any small Solar System body whose orbit brings it to proximity with Earth. This is a small website that shows NEOs passing by Earth today, previous, or next days. This website was developed in React, and the data are fetched from NASA via public APIs.
