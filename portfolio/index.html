Portfolio

Welcome to my portfolio, where you can find some interesting websites and program gadgets I have created on my own since I started learning programming.

Cycly

This is a cycle-renting website, where visitors can browse cycle brands, types, and models, and rent cycles if they sign up and log in. This full stack project uses Vue, Node, MongoDB for the front end, back end, database, respectively.

Click here to check it out

Milford Cinema

This is a cinema website where visitors can browse movies, and, if they sign up and log in, can leave comments and select seats. The technologies used in this web app are React, React-Redux, Firebase Realtime Database, Firebase Storage, etc.

Click here to check it out

Finding Neos

A near-Earth object (NEO) is any small Solar System body whose orbit brings it to proximity with Earth. This is a small website that shows NEOs passing by Earth today, previous, or next days. This website is developed in Angular, and the data are fetched from NASA via public APIs.

Click here to check it out

Game: Eric the Orange Boy

This game is programmed in vanilla JavaScript, HTML, and CSS, bundled by Webpack, and transpiled by Babel. The gamer can use the Arrow keys to let Eric move left, right, up, and down, in order to catch falling oranges. Eric would yay and grin if he catches an orange, and ouch (or uhoh) and frown if hit by an orange. Oranges appear randomly on the tree, and grow bigger and bigger from green to orange, and then hang there for a while before falling.
