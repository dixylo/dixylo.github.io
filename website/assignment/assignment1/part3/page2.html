How to use javascript

HTML, CSS and Javascript are a triplet that can create fantastic webpage based on the coordination of the three. As with CSS, javascript can also be embedded among lines of HTML code or be called from a separate javascript file. For the first case, like "" are used for CSS to function in HTML, javascript uses "" to denote a block of code that deals with the behavior of a webpage. It actually acts like a event handler that handles the event generated by a user, for example, clicking of a button, dragging of a slide, changing of text, etc. And it outputs a result, which can be the updating of the text of a component, playing a piece of music, changing the color of background, etc. In next page, we will see several demonstrations that show how javascript interacts with the user.
