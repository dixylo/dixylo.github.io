Chapter 1: Why JavaScript?

HTML is the language that programmers use to tell browsers what to display, whereas CSS is the one that deals with how to display. With these two languages, you are already able to create a decent, even beautiful website. However, it would only be a showcase, where everything is exhibited behind glass and not touchable. This lack of interaction would definitely limit the generation and flow of information. As a solution to the limitation, JavaScript is a powerful tool that allows the interaction between websites and users. It is a language that programmers use to program the behavior of webpages. With JavaScript, webpages now can response to the user that is browsing it, like recording the information the user inputs, open a link that the user clicks, etc. Now you have come to know JavaScript, please click Next to know more about it.
